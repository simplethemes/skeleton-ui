And Credits

While this package includes the full demo you see here, the SASS source (also requires Compass) is where it all is. If for any reason you don't want to load any specific piece of Skeleton UI, just remove its include from screen.scss

Skeleton UI uses some jQuery and a few lightweight plugins to help with various elements.
The links below give credit where it's due, as well as provide you with an explanation of the code found in this demo.

  • jQuery Um...
  • Modernizr Helps crappy browsers understand HTML5 (modernizr-2.6.2.min.js)
  • Simple Tabs Stupid Simple Tabs (main.js)
  • Superfish Menus Dropdown submenu navigation (jquery.superfish.js)
  • Formalize Cross-browser form consitency (jquery.formalize.js)
  • Lightbox.me Super lightweight modal popup (jquery.lightbox_me.js)
  • Mobile Nav Responsive Navigation (main.js)
  • Sticky Footer Stick the footer to the bottom of the page. (main.js)
  • Highlight.js (just for demo. remove it) (document head)
  • Phrozn Special thanks for providing a tool for the gh-pages/demo branch.
