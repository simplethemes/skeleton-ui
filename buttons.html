The SASS Way

Generating the buttons below is simple with SASS/Compass using the @simple-button mixin.

The simple-button mixin accepts three parameters: HSL. You can convert or get these values in your favorite image editor or use the handy HSL Color Picker

SASS Formula:.btn.orange { @include simple-button(35, 94%, 51%); }
The Result: Orange Button 
SASS Forumla:// Creates a .green button class with alias of .success 
.btn.green,.btn.success @include simple-button(85, 72%, 33%);
The Result: Green Button
